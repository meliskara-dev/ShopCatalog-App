# 🛍️ ShopCatalog App

#### Shop Catalog App is a Flutter application that allows users to browse products, view detailed information, and manage their shopping cart.
---

## 📱 Screens

| Screen | Description |
|---|---|
| 🔐 Welcome | Get started |
| 🛒 Product List | Product listing with GridView and searchbar |
| 📦 Product Detail | Product details with add to cart |
| 🧺 Cart | Cart management  |

## ⚙️ Getting Started

```bash
# Clone the repository
git clone https://github.com/meliskara-dev/ShopCatalog-App.git

# Navigate to the project directory
cd ShopCatalog-App

# Install dependencies
flutter pub get

# Run the app
flutter run
```
